STUFFER

Stuffer helps users track their belongings in an intelligent way.
• Simply ask Alexa to add an item. When prompted later, it informs the user of the same.
• Stuffer maintains a history of every item ever stored.
• Stuffer also tracks items required for an event. Ask Stuff Locator to create an event and then add items to the list. Later list an event out.

JOBSTER

Built using the nltk and textBlob libraries of Python and NodeJS, this app allows a user to score their resume based on a job description uploaded by a company. This helps them to improve their resume and make it more relevant for the given job description through various keyword matching scripts (run in Python) using text matching algorithms and NLP. This app also allows the user to share their resume on several job portals.

OORJA

This was built as a part of a 24 hr hackathon in which my team was placed 3rd among 54 participating teams. This website, available in 100+ languages, aims to support those rural youth who aspire to be a sport person but are unaware of the opportunities around them. It supports 10+ features, all of which can be found on the website itself. It runs on Node.Js and uses MongoDB, OAuth, Express.js, ScraperJs, gTranslate, etc.

BUMBA : YOUR WEB ASSISTANT

My first live project, manages the users' bookmarks effectively, maintains a multi-functional 'To Do List', checks latest g-mail notifications and users can doodle out important diagrams, figures, charts etc.
It runs on nodeJS and uses a multitude of web technologies including MongoDB, Oauth - Passport.js, Socket.io, Express.js, PhantomJS,Bootstrap 4 and jQuery , among others. You may check the demo.

TRACK YOUR TABS

Track-Your-Tabs is a chrome browser extension which helps you keep track of the tabs on your browser and also the time that you spend on the internet. You can check all your visits to a particular website too. It also gives you a graphical analysis of the data, so that the next time you can be more efficient. The app is in great progress and can be tested from the repo link.It uses a number of Chrome Extension APIs.

EXPENSE TRACKER

Built on Java, this project aims to help you keep track of your daily expenses. You may enter the reason, amount, mode of payment and it will automatically store it along with the date and time. Just sign up or make an account and you are ready to go. Once you sign in you can view all your stored transactions, delete the ones you no more wish to keep.
